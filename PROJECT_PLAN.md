# План проекта "ЧтоЕсть"

## Краткое описание

Приложение для планирования приемов пищи для семейной пары с локальной базой данных рецептов.

## Основной функционал

### 1. Выбор ингредиентов
- Пользователь выбирает один или несколько ингредиентов из списка
- Интуитивный интерфейс с поиском и фильтрацией по категориям
- Визуальное отображение выбранных ингредиентов

### 2. Подбор блюд
- После выбора ингредиентов приложение предлагает 3-5 вариантов блюд
- Блюда отображаются в виде карточек с изображением, названием и кратким описанием
- Алгоритм поиска учитывает все выбранные ингредиенты

### 3. Просмотр рецепта
- Детальный рецепт с пошаговыми инструкциями
- Список ингредиентов с точными количествами
- Информация о времени приготовления, сложности и количестве порций

## Технический стек

- **Frontend**: React 18 + TypeScript
- **UI**: Material-UI
- **State Management**: Redux Toolkit
- **База данных**: SQLite (sql.js для браузера)
- **Build Tool**: Vite
- **Стилизация**: Material-UI + Emotion

## Структура данных

### Основные сущности

1. **Ingredient (Ингредиент)**
   - id, name, category, image_url

2. **Dish (Блюдо)**
   - id, name, description, image_url, cooking_time, difficulty, servings

3. **Recipe (Рецепт)**
   - id, dish_id, instructions (JSON)

4. **RecipeIngredient (Ингредиент рецепта)**
   - recipe_id, ingredient_id, quantity, unit

## Алгоритм поиска блюд

1. Пользователь выбирает ингредиенты (например: курица, макароны)
2. Приложение ищет блюда, которые содержат:
   - Все выбранные ингредиенты (приоритет)
   - Большинство выбранных ингредиентов (вторичный приоритет)
3. Результаты сортируются по количеству совпадений
4. Возвращается максимум 5 блюд

## UI/UX концепция

### Главный экран
- Заголовок приложения
- Компонент выбора ингредиентов
- Кнопка "Найти блюда"

### Экран списка блюд
- Карточки блюд (3-5 штук)
- Каждая карточка содержит:
  - Изображение
  - Название
  - Краткое описание
  - Время приготовления
  - Сложность

### Экран рецепта
- Название блюда
- Изображение
- Информация (время, сложность, порции)
- Список ингредиентов с количествами
- Пошаговые инструкции
- Кнопка "Назад"
