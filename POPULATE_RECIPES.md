# Заполнение базы данных рецептами из TheMealDB

Этот скрипт получает 100 уникальных рецептов из TheMealDB, переводит их на русский через OpenAI и генерирует SQL для добавления в базу данных.

## Требования

1. **OpenAI API ключ** — должен быть в `.env` как `VITE_OPENAI_API_KEY`
2. **Node.js** — для запуска скрипта
3. **Интернет-соединение** — для запросов к TheMealDB и OpenAI

## Использование

```bash
node scripts/populate-recipes.mjs
```

## Что делает скрипт

1. **Получает рецепты** — делает запросы к TheMealDB API для получения 100 уникальных случайных рецептов
2. **Переводит на русский** — использует GPT-4o-mini для перевода названий, описаний, ингредиентов и инструкций
3. **Генерирует SQL** — создаёт файл `database/recipes-seed.sql` с INSERT statements для:
   - Ингредиентов (ingredients)
   - Блюд (dishes)
   - Связей блюд и ингредиентов (dish_ingredients)
   - Рецептов с инструкциями (recipes)
   - Ингредиентов рецептов с количеством (recipe_ingredients)

## Время выполнения

- Получение 100 рецептов: ~20-30 секунд
- Перевод через OpenAI: ~100 секунд (1 секунда на рецепт)
- **Итого: ~2-3 минуты**

## Результат

После выполнения скрипта:

1. Откройте файл `database/recipes-seed.sql`
2. Скопируйте его содержимое
3. Добавьте в конец файла `database/seed.sql`
4. Перезапустите приложение — новые рецепты появятся в базе

## Важно

- Скрипт использует GPT-4o-mini (~$0.0001 за рецепт), итого ~$0.01 за 100 рецептов
- Фото рецептов берутся из TheMealDB (постоянные URL)
- Категории ингредиентов определяются автоматически по названию
- Вегетарианство/веганство определяется OpenAI на основе ингредиентов

## Пример использования

```bash
# 1. Убедитесь, что .env содержит VITE_OPENAI_API_KEY
cat .env | grep VITE_OPENAI_API_KEY

# 2. Запустите скрипт
node scripts/populate-recipes.mjs

# 3. Проверьте результат
cat database/recipes-seed.sql

# 4. Добавьте в seed.sql
cat database/recipes-seed.sql >> database/seed.sql
```

## Устранение проблем

### Ошибка: VITE_OPENAI_API_KEY не найден
- Проверьте, что файл `.env` существует в корне проекта
- Убедитесь, что ключ указан как `VITE_OPENAI_API_KEY=sk-...`

### Ошибка: OpenAI API error
- Проверьте баланс на аккаунте OpenAI
- Убедитесь, что ключ действителен
- Проверьте интернет-соединение

### Мало рецептов получено
- TheMealDB может вернуть дубликаты — скрипт автоматически их фильтрует
- Если получено меньше 100, запустите скрипт ещё раз (он добавит новые)
